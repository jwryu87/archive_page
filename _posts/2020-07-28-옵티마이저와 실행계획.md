### 2. 실행계획

 SQL에서 요구한 사하을 처리하기 위한 절차와 방법을 의미. 
 실행계획을 생성한다는 것은 SQL을 어떤 순서로 어떻게 실행할 지를 결정하는 작업.
 
 옵티마이저가 ***최적의 실행계획***을 생성해 준다.
 
### 3. SQL 처리 흐름도

 실행계획의 시각화


```
* 실행계획 샘플

select    *
from      fne_eco_mast a
          inner join fne_eco_data b
          on (a.eco_cd = b.eco_cd) 
where     1=1
and       a.eco_cd in (
                      '13.03.007.003.005', 
                      '13.03.007.004.005', 
                      '13.03.007.006.001', 
                      '14.09.002.008',     
                      '14.09.002.009',     
                      '14.09.002.010'
                      )
and       b.trd_dt = '20200723'
order by 1
;
```
